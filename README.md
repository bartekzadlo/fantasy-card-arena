# fantasy-card-arena
Fantasy Card Arena (nazwa robocza) to turowa gra PvP oparta o system kart, w której gracze rywalizują przy użyciu własnych talii jednostek. Każda karta reprezentuje jednostkę posiadającą statystyki oraz umiejętności, które wpływają na przebieg walki.  Gra skupia się na symulacji starcia dwóch graczy, gdzie kluczowe znaczenie ma strategia.
